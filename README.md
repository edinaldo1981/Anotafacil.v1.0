## 📱 Sobre o App

O **Anota Fácil** é um sistema web progressivo (PWA) para gestão de estabelecimentos como açougues, conveniências e mercadinhos. Permite controle de vendas, clientes, colaboradores e finanças, com envio automático de comprovantes via WhatsApp.

---

## 👥 Perfis de Acesso

| Perfil | Acesso |
|--------|--------|
| 🔐 Super Admin | Painel da plataforma — cadastra estabelecimentos e donos |
| 🏪 Dono | Gestão completa do estabelecimento |
| 👤 Colaborador | Nova Compra, Dashboard e Histórico |

---

## 🛠 Stack Técnica

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express
- **IA:** Gemini API
- **Deploy:** Docker + Nginx
- **PWA:** Service Worker + Web Manifest

---

## 🚀 Deploy Rápido

```bash
git clone https://github.com/edinaldo1981/Aplicativo-para-conveniencias.git
cd Aplicativo-para-conveniencias
docker-compose up -d --build
```

Acesse: `http://localhost`

## 📋 Funcionalidades

- ✅ Login com 3 níveis de acesso
- ✅ Dashboard com métricas em tempo real
- ✅ Cadastro e gestão de clientes
- ✅ Nova Compra com catálogo de produtos
- ✅ Preview de comprovante para WhatsApp e impressão
- ✅ Histórico de compras com filtros
- ✅ Gestão de colaboradores com controle de atendimento
- ✅ Observações por venda (recados do cliente)
- ✅ Módulo financeiro — a prazo, juros, inadimplentes
- ✅ Configurações (promoções e produtos)
- ✅ Painel Super Admin com MRR, alertas e controle de assinaturas
- ✅ Instalável como app (PWA)
